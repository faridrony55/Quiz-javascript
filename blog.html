Question Answers:

  • What are difference between Map, filter, foreach, find ?

    map() create a new array from the previous data without modifying any data

    filter() pass the value and get the data from array and make a new array and return the value.

    forEach() exicute the data without creating an array and performing some actions on each element

    find() return the specific value based on requested value

  • What are difference between local storage, session storage , cookies ?

    local Storage: use to store data on the client’s computer. Allow saving of values in a web browser. There are no expiration date. User can clear the browser data or cache to erase all localStorage data. And data is readable

    session Storage: stores the data only for a session, meaning that the data is stored until the browser is closed

    cookies: a small pieces of data stored in client’s computer. The data such as user preferences, login details, and shopping cart items are stored.

  • How event loop works?

    Event Loop use to monitor callstack and callback queue and processing them one by one.
